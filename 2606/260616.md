# 1. Native Query

## 개념
- JPA에서 SQL을 직접 사용하는 기능
- ORM 기능을 유지하면서 DB 고유 기능이나 복잡한 쿼리를 사용할 수 있음

### 사용하는 경우
- JPQL로 표현하기 어려운 복잡한 쿼리
- DBMS 전용 함수 사용
- 성능 최적화가 필요한 경우

---

## Native Query 생성 방법

### 1. 결과 타입 지정
```java
createNativeQuery(sql, Menu.class)
```

- 조회 결과를 엔티티 객체로 반환
- 모든 컬럼을 조회해야 함

### 예시
```java
SELECT menu_code,
       menu_name,
       menu_price,
       category_code,
       orderable_status
FROM tbl_menu
WHERE menu_code = ?
```

결과
```java
Menu menu
```

---

### 2. 결과 타입 지정 불가

일부 컬럼만 조회할 경우 사용

```java
createNativeQuery(sql)
```

### 예시

```sql
SELECT menu_name, menu_price
FROM tbl_menu
```

결과

```java
List<Object[]>
```

```java
row[0] -> menu_name
row[1] -> menu_price
```

---

## 결과 매핑(Result Mapping)

### 자동 매핑

`@Column` 정보가 존재할 때 사용

```java
@SqlResultSetMapping(
    name = "categoryCountAutoMapping",
    entities = {
        @EntityResult(entityClass = Category.class)
    },
    columns = {
        @ColumnResult(name = "menu_count")
    }
)
```

### 특징

- 엔티티 자동 매핑
- 추가 컬럼은 `@ColumnResult` 사용

---

### 수동 매핑

컬럼명과 엔티티 필드명이 다를 경우 사용

```java
@FieldResult(
    name = "categoryCode",
    column = "category_code"
)
```

### 특징

- 컬럼과 필드를 직접 연결
- 자동 매핑이 어려울 때 사용

---

## 자동 매핑 vs 수동 매핑

| 구분 | 자동 매핑 | 수동 매핑 |
|--------|--------|--------|
| 설정 | 간단 | 복잡 |
| 사용 조건 | @Column 존재 | 직접 지정 필요 |
| 추천 | 가능하면 사용 | 특수 케이스 |

# 2. Named Native Query

## 개념

미리 SQL을 정의해두고 이름으로 호출하는 방식

```java
@NamedNativeQuery
```

---

## 장점

- SQL 재사용 가능
- 컴파일 시점 검증 가능
- 코드 가독성 향상
- 유지보수 편리

---

## 정의

```java
@NamedNativeQuery(
    name = "Category.menuCountOfCategory",
    query = "...",
    resultSetMapping = "categoryCountAutoMapping2"
)
```

---

## 실행

```java
Query query =
    manager.createNamedQuery(
        "Category.menuCountOfCategory"
    );
```

---

## Native Query 정리

### 조회 결과가 엔티티 전체

```java
createNativeQuery(sql, Entity.class)
```

### 조회 결과가 일부 컬럼

```java
createNativeQuery(sql)
```

결과

```java
Object[]
```

### 엔티티 + 추가 컬럼

```java
@SqlResultSetMapping
```

### 자주 사용하는 SQL

```java
@NamedNativeQuery
```
