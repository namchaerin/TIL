# Chap07. Spring Data JPA

## 1. Spring Data JPA 개요

### 1.1 Spring Data JPA란?
- Spring에서 JPA를 더 쉽게 사용할 수 있도록 지원하는 모듈
- Repository 인터페이스를 통해 CRUD 처리
- 메서드 이름만으로 SQL/JPQL 생성 가능

### 1.2 특징
- Repository 인터페이스 제공
- Query Method 지원
- Querydsl 연동 가능
- Pagination 지원
- EntityManagerFactory, EntityManager, EntityTransaction 직접 사용 불필요

---

## 2. Repository 인터페이스

### 2.1 상속 구조

| 인터페이스 | 설명 |
|-----------|------|
| Repository | 최상위 인터페이스 |
| CrudRepository | CRUD 기능 제공 |
| PagingAndSortingRepository | 페이징 및 정렬 지원 |
| JpaRepository | JPA 관련 기능 추가 제공 |

### 2.2 주요 메서드

| 메서드 | 설명 |
|---------|------|
| count() | 전체 개수 조회 |
| findById() | PK 조회 |
| findAll() | 전체 조회 |
| save() | 저장 및 수정 |
| saveAll() | 다건 저장 |
| delete() | 삭제 |
| existsById() | 존재 여부 확인 |

---

## 3. Query Method

### 3.1 Query Method란?
- 메서드 이름만으로 JPQL 자동 생성
- 형식: `findBy필드명()`

예시
```java
findByCode()
findByName()
findMenuByCode()
```

---

### 3.2 주요 키워드

| 키워드 | 예시 |
|---------|------|
| And | findByCodeAndName |
| Or | findByCodeOrName |
| Between | findByPriceBetween |
| LessThan | findByPriceLessThan |
| GreaterThan | findByPriceGreaterThan |
| IsNull | findByNameIsNull |
| IsNotNull | findByNameIsNotNull |
| Like | findByNameLike |
| StartingWith | findByNameStartingWith |
| EndingWith | findByNameEndingWith |
| Containing | findByNameContaining |
| OrderBy | findByPriceOrderByCodeDesc |
| In | findByNameIn |

### 예시

```java
findByCodeAndName()
findByPriceBetween()
findByNameContaining()
findByNameIn(List<String> names)
findByPriceOrderByCodeDesc()
```
