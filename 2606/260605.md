# MyBatis XML 설정 및 Mapper 활용

## 1. MyBatis Config XML

### 1-1. mybatis-config.xml 역할

- MyBatis의 전역 설정을 관리하는 파일이다.
- DB 연결 정보, Mapper 등록, 별칭(TypeAlias) 등을 설정한다.

### 1-2. 주요 설정 태그

#### 1-2-1. `<properties>`

- DB 접속 정보 등의 공통 설정을 외부 properties 파일로 분리한다.
- `${key}` 형식으로 값을 참조한다.

#### 1-2-2. `<settings>`

- MyBatis의 동작 방식을 설정한다.
- 대표 설정
    - mapUnderscoreToCamelCase
    - cacheEnabled
    - defaultExecutorType

#### 1-2-3. `<typeAliases>`

- DTO 클래스에 별칭을 부여한다.
- 긴 패키지명을 줄여 Mapper에서 간편하게 사용할 수 있다.

#### 1-2-4. `<environments>`

- 데이터베이스 연결 정보를 설정한다.
- 트랜잭션 관리 방식과 DataSource를 지정한다.

#### 1-2-5. `<mappers>`

- Mapper XML 파일을 등록한다.
- MyBatis가 SQL을 찾을 수 있도록 연결해준다.

## 2. Mapper Interface 활용

### 2-1. Mapper 역할

- SQL 실행을 위한 인터페이스이다.
- SqlSession이 구현체를 생성해준다.

```java
Mapper mapper = sqlSession.getMapper(Mapper.class);
```

### 2-2. 주요 어노테이션

#### 2-2-1. @Select

- 조회(SELECT) SQL 작성

#### 2-2-2. @Insert

- 데이터 등록(INSERT)

#### 2-2-3. @Update

- 데이터 수정(UPDATE)

#### 2-2-4. @Delete

- 데이터 삭제(DELETE)

#### 2-2-5. @Results / @Result

- 조회 결과와 DTO 필드를 매핑한다.

```java
@Result(property = "name", column = "MENU_NAME")
```

## 3. Mapper XML 작성

### 3-1. Mapper XML 역할

- SQL 쿼리와 결과 매핑 정보를 작성한다.
- CRUD 쿼리를 XML로 관리할 수 있다.

### 3-2. 작성 규칙

- Mapper 인터페이스와 동일한 패키지 구조를 사용한다.
- namespace는 Mapper 인터페이스의 전체 경로와 일치해야 한다.
- 인터페이스 메서드명과 XML의 id가 동일해야 한다.

---

## 4. 주요 Mapper 태그

### 4-1. `<resultMap>`

- 조회 결과를 DTO 객체에 매핑한다.

```xml
<resultMap id="menuResultMap" type="MenuDTO">
    <id property="code" column="MENU_CODE"/>
    <result property="name" column="MENU_NAME"/>
</resultMap>
```

### 4-2. `<select>`

- 조회 SQL을 작성한다.
- resultType 또는 resultMap을 사용해 결과를 매핑한다.

```xml
<select id="selectMenuByCode" resultMap="menuResultMap">
    SELECT * FROM TBL_MENU
</select>
```

### 4-3. `<insert>, <update>, <delete>`

- 데이터 등록, 수정, 삭제 SQL을 작성한다.
- 수행 결과로 영향받은 행(row) 수를 반환한다.

```xml
<insert id="insertMenu">
    INSERT INTO TBL_MENU ...
</insert>
```

### 4-4. `<cache>`

- 조회 결과를 캐시에 저장한다.
- 반복 조회 시 성능 향상에 도움이 된다.

### 4-5. `<sql>`

- 공통 SQL 조각을 정의하고 재사용한다.

```xml
<sql id="menuColumns">
    MENU_CODE,
    MENU_NAME,
    MENU_PRICE
</sql>
```

```xml
<select id="selectMenu">
    SELECT
    <include refid="menuColumns"/>
    FROM TBL_MENU
</select>
```
