# HTML & CSS 핵심 정리

## 1. HTML (HyperText Markup Language)

### **텍스트 및 목록 태그**
* **제목 태그**: `h1`부터 `h3`까지 사용하여 문서의 구조적인 제목을 표시한다.
* **문단 및 서식**:
    * `p`: 일반적인 본문 문단을 만든다.
    * `br`: 문단 내에서 줄바꿈을 수행한다.
    * `hr`: 수평선을 그어 주제를 구분한다.
    * `pre`: 코드나 서식이 중요한 문장을 작성한 그대로 표시한다.
    * `strong` / `em`: 중요한 내용을 강조하거나 기울여 표시한다.
* **목록**:
    * `ul`: 순서가 없는 목록(불렛 포인트)을 만든다.
    * `ol`: 순서가 있는 목록(숫자)을 만든다.
    * `li`: 각 목록의 개별 항목을 정의한다.

### **테이블과 미디어 태그**
* **표(Table)**:
    * `table`: 표의 전체 영역을 감싼다.
    * `tr`: 표의 한 행(Row)을 정의한다.
    * `th`: 제목 셀로, 굵게 표시된다.
    * `td`: 일반 데이터 셀을 정의한다.
    * `rowspan` / `colspan`: 셀을 세로 또는 가로로 합칠 때 사용한다.
* **미디어**:
    * `img`: 이미지를 삽입하며 `alt` 속성으로 대체 텍스트를 제공한다.
    * `audio` / `video`: 오디오와 비디오 파일을 삽입하며 `controls` 속성으로 재생 도구를 표시한다.

### **링크 및 폼(Form)**
* **링크**: `a` 태그와 `href` 속성을 사용하여 외부 사이트나 내부 파일, 페이지 내 특정 위치로 이동한다.
* **폼(Form)**:
    * `form`: 사용자 데이터를 서버로 전송하기 위한 창구 역할을 한다.
    * `input`: 텍스트, 비밀번호, 라디오 버튼, 체크박스 등 다양한 입력을 받는다.
    * `label`: `id`를 통해 입력 요소와 연결하여 클릭 편의성을 높인다.
    * `textarea`: 여러 줄의 텍스트를 입력받을 때 사용한다.
    * `button`: 작성 완료(`submit`)나 다시 작성(`reset`) 기능을 수행한다.

### **시맨틱 레이아웃**
* **일반 영역**: `div`(블록 레벨)와 `span`(인라인 요소)으로 영역을 나눈다.
* **의미론적 태그**: `header`, `main`, `section`, `footer` 등을 사용하여 웹 페이지의 구조를 명확히 정의한다.

## 2. CSS (Cascading Style Sheets)

### **선택자 (Selectors)**
* **기본 선택자**: 태그 이름, `.클래스명`, `#아이디명`을 사용하여 스타일 대상을 지정한다.
* **관계 선택자**:
    * **후손 선택자**: 띄어쓰기를 사용하여 특정 요소 내부의 모든 요소를 선택한다.
    * **자손 선택자**: `>`를 사용하여 바로 아래 자식 요소만 선택한다.

### **박스 모델 및 텍스트 스타일**
* **폰트 및 색상**: `font-family`, `font-size`, `color`, `text-align` 등으로 글꼴과 정렬을 조절한다.
* **박스 모델**:
    * `width` / `height`: 요소의 크기를 지정한다.
    * `padding`: 테두리 안쪽 여백을 설정한다.
    * `border`: 테두리 선의 두께와 종류를 설정한다.
    * `margin`: 요소 사이의 바깥 여백을 설정한다.
    * `border-radius`: 모서리를 둥글게 만든다.

### **레이아웃 기술**
* **Flexbox**: `display: flex`를 선언하고 `justify-content`, `align-items`로 요소를 유연하게 정렬한다.
* **Grid**: `display: grid`와 `grid-template-columns`를 사용하여 격자 구조를 만든다.
* **Position**: `relative`, `absolute`를 활용하여 요소를 원하는 위치에 배치한다.

### **인터랙션 및 효과**
* **가상 클래스**: `:hover`(마우스 오버), `:active`(클릭 중) 상태에 스타일을 적용한다.
* **애니메이션**: `transition` 속성을 사용하여 상태 변화를 부드럽게 표현한다.

## 느낌점 및 활용 방안
UI가 어떻게 생성되고 동작되는지 알 수 있었다. 개인 프로젝트 진행시에 UI를 직접 생성하는 경우에 활용해 볼 수 있을 것 같다.
