## 자바: 반복문과 반복제어문
### while
- 조건식이 true인 동안 반복 수행, 반복 횟수 미정일 때 사용, 조건을 먼저 검사 후 실행
```java 
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
```

### for
- 초기값·조건식·증감식으로 반복 제어, 반복 횟수가 정해진 경우 주로 사용
```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

#### break
- 반복문을 즉시 종료하고 바깥으로 탈출, 특정 조건에서 반복 중단할 때 사용
```java
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // i가 5가 되는 순간 반복문 전체 종료
    }
    System.out.println(i);  // 0~4까지만 출력됨
}
```
#### continue
- 현재 반복을 건너뛰고 다음 반복으로 진행, 특정 조건만 제외하고 반복 수행할 때 사용
```java
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue;   // i가 5일 때는 아래 코드 실행 안 하고 다음 반복으로 넘어감
    }
    System.out.println(i);  // 5만 제외하고 0~9 출력됨
}
```
