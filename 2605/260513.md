# Java: Enum, Lambda, Stream

# 1. Enum

## 1.1 Enum 개념
Enum은 서로 관련된 상수들을 하나의 타입으로 관리하기 위한 기능이다.  
상수를 의미 있는 이름으로 관리할 수 있어 코드의 가독성과 안정성을 높일 수 있다.

```java
public enum UserRole {
    ADMIN,
    USER,
    GUEST
}
```

---

## 1.2 Enum 특징
- 상수를 그룹화하여 관리 가능
- 타입 안정성 제공
- 문자열 대신 enum 타입 사용 가능
- switch문과 함께 자주 사용

---

## 1.3 Enum 사용 이유
기존 문자열이나 숫자로 상수를 관리하면 오타나 잘못된 값 입력 가능성이 존재한다.  
Enum을 사용하면 정해진 값만 사용할 수 있어 유지보수성과 안정성이 향상된다.

---

# 2. Lambda

## 2.1 Lambda 개념
람다식(Lambda Expression)은 메소드를 하나의 식으로 표현하는 방식이다.  
불필요한 코드 작성을 줄이고 간결한 함수형 프로그래밍 스타일을 지원한다.

```java
(x, y) -> x + y
```

---

## 2.2 Lambda 특징
- 익명 함수 형태로 사용
- 코드 간결화 가능
- 함수형 인터페이스와 함께 사용
- 컬렉션 및 Stream API와 자주 활용

---

## 2.3 Lambda 사용 예시

### 기존 방식

```java
Runnable runnable = new Runnable() {
    @Override
    public void run() {
        System.out.println("실행");
    }
};
```

### Lambda 방식

```java
Runnable runnable = () -> System.out.println("실행");
```

---

# 3. Stream

## 3.1 Stream 개념
Stream은 컬렉션 데이터를 함수형 방식으로 처리하기 위한 기능이다.  
반복문 없이 데이터 검색, 변환, 정렬 등을 수행할 수 있다.

---

## 3.2 Stream 특징
- 내부 반복 처리
- 코드 간결화
- 함수형 프로그래밍 지원
- 병렬 처리 가능

---

## 3.3 Stream 주요 메소드

### filter()
조건에 맞는 데이터만 추출

```java
list.stream()
    .filter(num -> num > 10);
```

---

### map()
데이터 변환

```java
list.stream()
    .map(String::toUpperCase);
```

---

### forEach()
데이터 반복 처리

```java
list.stream()
    .forEach(System.out::println);
```

---

### collect()
처리 결과를 리스트 등으로 변환

```java
List<String> result =
    list.stream()
        .collect(Collectors.toList());
```

---

# 4. Lambda와 Stream 관계

Stream은 Lambda와 함께 자주 사용된다.  
람다식을 활용하면 Stream 내부의 데이터 처리 로직을 간결하게 작성할 수 있다.

```java
list.stream()
    .filter(name -> name.startsWith("김"))
    .forEach(System.out::println);
```
