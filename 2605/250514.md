# sql: SQL 문법 정리

## 1. SELECT 문

### 1.1 SELECT
테이블에서 원하는 데이터를 조회하기 위한 기본 문법이다.

```sql
SELECT column_name
FROM table_name;
```

전체 컬럼 조회 시 `*`를 사용할 수 있다.

```sql
SELECT *
FROM tbl_menu;
```

---

## 2. WHERE 조건문

### 2.1 비교 연산자
특정 조건에 맞는 데이터만 조회할 때 사용한다.

```sql
WHERE menu_price < 10000
```

사용 예시
- `=` : 같다
- `!=`, `<>` : 같지 않다
- `>` : 크다
- `<` : 작다
- `>=`, `<=`

---

### 2.2 AND / OR

#### AND
모든 조건을 만족해야 한다.

```sql
WHERE menu_name LIKE '%밥%'
AND menu_price BETWEEN 20000 AND 30000
```

#### OR
여러 조건 중 하나만 만족해도 된다.

```sql
WHERE menu_price < 10000
OR menu_name LIKE '%김치%'
```

---

## 3. NULL 처리

### 3.1 IS NULL / IS NOT NULL

NULL 여부를 확인할 때 사용한다.

```sql
WHERE ref_category_code IS NOT NULL
```

NULL 비교 시 `=` 연산자는 사용할 수 없다.

---

## 4. LIKE 문

### 4.1 문자열 검색

특정 문자열 포함 여부를 검색할 때 사용한다.

```sql
WHERE menu_name LIKE '%김치%'
```

#### 와일드카드
- `%` : 모든 문자
- `_` : 한 글자

---

## 5. BETWEEN

범위 조건 조회 시 사용한다.

```sql
WHERE menu_price BETWEEN 20000 AND 30000
```

이상 ~ 이하 범위를 의미한다.

---

## 6. IN / NOT IN

여러 값을 한 번에 비교할 때 사용한다.

```sql
WHERE category_name NOT IN ('기타', '쥬스', '커피')
```

---

## 7. ORDER BY

조회 결과를 정렬할 때 사용한다.

```sql
ORDER BY category_name DESC
```

#### 정렬 방식
- ASC : 오름차순
- DESC : 내림차순

다중 정렬도 가능하다.

```sql
ORDER BY menu_price ASC, menu_name DESC
```

---

## 8. LIMIT

조회 결과 개수를 제한할 때 사용한다.

```sql
LIMIT 3
```

상위 N개 데이터 조회 시 자주 사용된다.

---

## 9. JOIN

### 9.1 JOIN 개념
두 개 이상의 테이블을 연결하여 조회할 때 사용한다.

---

### 9.2 INNER JOIN

연결 조건이 일치하는 데이터만 조회한다.

```sql
SELECT *
FROM employee e
JOIN job j ON e.job_code = j.job_code;
```

---

### 9.3 LEFT JOIN

왼쪽 테이블의 모든 데이터를 기준으로 조회한다.

```sql
SELECT *
FROM employee e
LEFT JOIN department d
ON d.dept_id = e.dept_code;
```

매칭되는 데이터가 없어도 왼쪽 테이블 데이터는 출력된다.

---

## 10. SELF JOIN

같은 테이블을 자기 자신과 JOIN 하는 방식이다.

직원과 관리자 관계 조회 시 사용된다.

```sql
FROM EMPLOYEE E
JOIN EMPLOYEE E2
ON E.MANAGER_ID = E2.EMP_ID
```

---

## 11. GROUP BY

같은 데이터를 그룹화할 때 사용한다.

```sql
GROUP BY d.dept_title
```

부서별 집계 처리에 활용된다.

---

## 12. 집계 함수

### 12.1 COUNT()
데이터 개수 조회

```sql
COUNT(*)
```

---

### 12.2 SUM()
합계 계산

```sql
SUM(e.salary)
```

---

### 12.3 AVG()
평균 계산

```sql
AVG(e.salary)
```

---

## 13. WITH ROLLUP

그룹별 결과와 전체 합계를 함께 출력할 때 사용한다.

```sql
GROUP BY d.dept_title
WITH ROLLUP;
```

---

## 14. 컬럼 별칭(ALIAS)

컬럼 이름을 변경하여 출력할 수 있다.

```sql
SELECT d.dept_title AS '부서명'
```

조회 결과 가독성을 높일 때 사용한다.
