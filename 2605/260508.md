# Java: Object 클래스와 String 클래스

## 1. Object 클래스

자바의 모든 클래스는 Object 클래스를 상속받는다.  
따라서 모든 객체는 `toString()`, `equals()`, `hashCode()` 등의 메소드를 사용할 수 있다.

---

## 2. toString()

`toString()`은 객체 정보를 문자열 형태로 반환하는 메소드이다.

기본적으로는 `클래스명@해시코드` 형태로 출력되며, 객체의 필드값을 보기 쉽게 출력하기 위해 오버라이딩하여 사용한다.

```java
@Override
public String toString() {
    return "Book{" +
            "number=" + number +
            ", title='" + title + '\'' +
            ", author='" + author + '\'' +
            ", price=" + price +
            '}';
}
```

---

## 3. equals()와 hashCode()

### equals()
객체의 주소값이 아닌 내부 데이터가 같은지 비교하기 위해 사용한다.

```java
@Override
public boolean equals(Object o) {

    if (o == null || getClass() != o.getClass()) return false;

    Book book = (Book) o;

    return number == book.number
            && price == book.price
            && Objects.equals(title, book.title)
            && Objects.equals(author, book.author);
}
```

---

### hashCode()
객체를 정수 형태의 해시값으로 변환하는 메소드이다.

```java
@Override
public int hashCode() {
    return hash(number, title, author, price);
}
```

equals() 결과가 true이면 hashCode() 값도 반드시 같아야 한다.

---

## 4. HashSet 동작 방식

HashSet은 다음 순서로 객체를 비교한다.

1. hashCode() 비교
2. equals() 비교

따라서 equals()와 hashCode()를 함께 오버라이딩해야 정상적으로 동작한다.

```java
Set<Book> books = new HashSet<>();

books.add(book2);

System.out.println(books.contains(book3));
```

# 5. String 클래스

String 클래스는 문자열 데이터를 처리하기 위한 클래스이다.

---

## 6. 문자열 처리 메소드

### trim()
문자열 앞뒤 공백 제거

```java
String trimmedData = rawData.trim();
```

### toUpperCase()
문자열을 대문자로 변환

```java
String upperData = trimmedData.toUpperCase();
```

### indexOf()
특정 문자 위치 검색

```java
int codeStart = upperData.indexOf('[');
```

존재하지 않으면 `-1` 반환

---

### substring()
문자열 일부 추출

```java
String productCode = upperData.substring(codeStart, codeEnd);
```

---

### replace()
특정 문자열 치환

```java
String replaced =
        productName.replace(
                "APPLE IPHONE 17 PRO",
                "GALAXY S26 ULTRA"
        );
```

---

## 7. 문자열 비교

### ==
문자열 주소값 비교

### equals()
문자열 내용 비교

```java
System.out.println(str1.equals(str2));
```

문자열 비교 시에는 equals()를 사용하는 것이 일반적이다.

---

## 8. String의 불변성

String 객체는 한 번 생성되면 수정할 수 없다.

```java
str2 += " mysql";
```

기존 문자열을 수정하는 것이 아니라 새로운 객체가 생성된다.

---

## 9. split()

구분자를 기준으로 문자열을 나누는 메소드이다.

```java
String[] empArr = emp.split("/");
```

배열 형태로 반환되어 반복문과 함께 사용된다.
