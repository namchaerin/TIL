# Java: 입출력(IO)과 파일 처리

## 1. 입출력(IO)

### 1.1 입출력(IO)이란?
IO(Input / Output)는 프로그램과 외부 장치 간 데이터를 주고받기 위한 기능이다.  
사용자 입력, 화면 출력, 파일 저장 등의 작업에 사용된다.

---

### 1.2 스트림(Stream)

스트림은 데이터를 읽고 쓰기 위한 단방향 통로이다.

- 입력 스트림 : 데이터 읽기
- 출력 스트림 : 데이터 쓰기

스트림은 기본적으로 1바이트 단위로 처리된다.

---

### 1.3 스트림 종류

#### 바이트 단위 스트림
영어, 숫자, 특수문자 처리에 사용된다.

| 종류 | 설명 |
| --- | --- |
| InputStream | 바이트 입력 스트림 |
| OutputStream | 바이트 출력 스트림 |

---

#### 문자 단위 스트림
한글과 같은 문자 데이터를 처리할 때 사용된다.

| 종류 | 설명 |
| --- | --- |
| Reader | 문자 입력 스트림 |
| Writer | 문자 출력 스트림 |

문자 단위 스트림은 인코딩 방식을 고려하여 데이터를 처리한다.

---

### 1.4 파일 입출력을 사용하는 이유

- 사용자 입력 및 출력 처리
- 파일 형태로 데이터 저장 가능
- 프로그램 종료 후에도 데이터 유지 가능

---

## 2. File 클래스

### 2.1 File 클래스란?
파일 및 디렉토리를 관리하기 위한 클래스이다.

파일 생성, 삭제, 경로 조회 등의 기능을 제공한다.

```java
File file = new File("C:/data/test.txt");
```

---

### 2.2 파일 생성 및 삭제 메소드

| 메소드 | 설명 |
| --- | --- |
| createNewFile() | 파일 생성 |
| mkdir() | 디렉토리 생성 |
| mkdirs() | 상위 경로까지 모두 생성 |
| delete() | 파일/디렉토리 삭제 |

---

### 2.3 파일 정보 확인 메소드

| 메소드 | 설명 |
| --- | --- |
| getName() | 파일 이름 조회 |
| getPath() | 전체 경로 조회 |
| getParent() | 부모 경로 조회 |
| isFile() | 파일 여부 확인 |
| isDirectory() | 디렉토리 여부 확인 |
| length() | 파일 크기 조회 |
| lastModified() | 수정 시간 조회 |
| canRead() | 읽기 가능 여부 |
| canWrite() | 쓰기 가능 여부 |

---

## 3. 기반 스트림

기반 스트림은 실제 데이터 입출력을 담당하는 스트림이다.


### 3.1 바이트 기반 스트림

| 스트림 | 설명 |
| --- | --- |
| FileInputStream | 파일 읽기 |
| FileOutputStream | 파일 쓰기 |

```java
FileInputStream fis = new FileInputStream("test.txt");
FileOutputStream fos = new FileOutputStream("test.txt");
```

---

### 3.2 문자 기반 스트림

| 스트림 | 설명 |
| --- | --- |
| FileReader | 문자 파일 읽기 |
| FileWriter | 문자 파일 쓰기 |

```java
FileReader fr = new FileReader("test.txt");
FileWriter fw = new FileWriter("test.txt");
```

---

## 4. 보조 스트림

보조 스트림은 기반 스트림 기능을 확장하거나 성능을 향상시키기 위해 사용된다.
보조 스트림 단독 사용은 불가능하며 반드시 기반 스트림과 함께 사용해야 한다.


### 4.1 Buffered 계열

입출력 성능 향상을 위한 보조 스트림이다.

| 스트림 | 설명 |
| --- | --- |
| BufferedInputStream | 입력 속도 향상 |
| BufferedOutputStream | 출력 속도 향상 |
| BufferedReader | 문자 입력 성능 향상 |
| BufferedWriter | 문자 출력 성능 향상 |

```java
BufferedReader br =
        new BufferedReader(new FileReader("test.txt"));
```

---

### 4.2 형변환 보조 스트림

바이트 스트림과 문자 스트림을 연결한다.

| 스트림 | 설명 |
| --- | --- |
| InputStreamReader | 바이트 → 문자 변환 |
| OutputStreamWriter | 문자 → 바이트 변환 |

한글 깨짐 방지를 위해 사용된다.

---

### 4.3 데이터 입출력 스트림

기본 자료형 데이터를 처리하기 위한 보조 스트림이다.

| 스트림 | 설명 |
| --- | --- |
| DataInputStream | 데이터 읽기 |
| DataOutputStream | 데이터 쓰기 |

```java
DataOutputStream dos =
        new DataOutputStream(new FileOutputStream("data.dat"));
```

---

### 4.4 객체 입출력 스트림

객체 단위 데이터를 저장하거나 읽을 때 사용한다.

| 스트림 | 설명 |
| --- | --- |
| ObjectInputStream | 객체 읽기 |
| ObjectOutputStream | 객체 쓰기 |

```java
ObjectOutputStream oos =
        new ObjectOutputStream(new FileOutputStream("object.dat"));
```
