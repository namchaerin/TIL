# 자바: 정렬(Sort)

## 1. 데이터 교환 (Swap)

* 두 변수 또는 배열 요소 값 교환 방법 학습
* temp 변수를 활용한 값 이동 방식 이해
* 수행 순서: temp = a → a = b → b = temp
* 배열에서의 swap 적용 방법 확인

```java
int temp = arr[0];
arr[0] = arr[1];
arr[1] = temp;
```

## 2. 정렬 알고리즘 (Sorting Algorithms)

### ① 선택 정렬 (Selection Sort)

* 최솟값을 선택하여 앞쪽으로 배치하는 방식 이해
* 전체 탐색을 통해 최소값 인덱스 탐색
* 현재 위치와 최소값 위치 swap 수행

```java
for (int i = 0; i < arr.length - 1; i++) {
    int minIndex = i; // 최소값 인덱스 가정
    for (int j = i + 1; j < arr.length; j++) {
        if (arr[minIndex] > arr[j]) {
            minIndex = j; // 더 작은 값 발견 시 갱신
        }
    }
    int temp = arr[i];
    arr[i] = arr[minIndex];
    arr[minIndex] = temp; // 최소값과 현재 위치 교환
}
```

---

### ② 버블 정렬 (Bubble Sort)

* 인접한 요소 비교 후 큰 값을 뒤로 이동시키는 방식 이해
* 한 회차 종료 시 가장 큰 값이 뒤로 이동됨
* 반복 수행으로 정렬 완성

```java
for (int i = arr.length - 1; i >= 0; i--) { // 정렬 확정 범위 축소
    for (int j = 0; j < i; j++) {
        if (arr[j] > arr[j + 1]) { // 앞이 크면 교환
            int temp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = temp; // swap 수행
        }
    }
}
```

---

### ③ Arrays.sort()

* 자바 기본 제공 정렬 라이브러리 사용 방법 학습
* 내부적으로 효율적인 알고리즘 사용
* 간단한 호출로 정렬 수행 가능

```java
import java.util.Arrays;

Arrays.sort(arr);
```

## 3. 응용: 중복 없는 로또 번호 생성

* 배열, 난수, 중복 검사, 정렬 종합 활용
* 배열 선언: 6개 정수 저장
* 난수 생성 후 중복 여부 검사
* 중복 발생 시 재추첨 처리
* 정렬 후 결과 출력

```java
import java.util.Arrays;

int[] lotto = new int[6];

for (int i = 0; i < lotto.length; i++) {
    int num = (int)(Math.random() * 45) + 1;

    for (int j = 0; j < i; j++) {
        if (lotto[j] == num) {
            i--; // 중복 발생 → 다시 추첨
            break;
        }
    }
    lotto[i] = num;
}

Arrays.sort(lotto); // 결과 정렬

for (int n : lotto) {
    System.out.print(n + " ");
}
```
